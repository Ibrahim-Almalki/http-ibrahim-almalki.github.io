PortSweep Recon

A high-speed network reconnaissance project utilizing Masscan for rapid port discovery across target IP ranges. This initiative focuses on efficiently identifying open ports and services on a network, providing critical first intelligence for penetration testing and security assessments. The project demonstrates the power of asynchronous scanning to quickly map accessible endpoints like FTP (21), SSH (22), Telnet (23), SMTP (25), DNS (53), and HTTP (80).

ServiceMapper Pro

A deep-dive service enumeration and OS fingerprinting project using Nmap for comprehensive network reconnaissance. This initiative focuses on advanced scanning techniques to precisely identify running services, their versions, and underlying operating systems on a target host. The project reveals a complete attack surface, from common web servers (Apache) and databases (MySQL, PostgreSQL) to legacy and potentially vulnerable services (Samba, FTP, Telnet, IRC), providing critical intelligence for vulnerability assessment and penetration testing.
